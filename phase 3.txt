# 🎨 SSTPSR Project — Phase 3 (Design System + UI/UX Blueprint)

---

# 🎯 OBJECTIVE

Ek **premium, modern, trust-building design system** create karna jo:

* parents ko impress kare
* admissions ko boost kare
* professional boarding school feel de
* mobile-first ho
* clean aur structured ho

---

# 🧠 PHASE 3 CORE OUTPUTS

Phase 3 ke end par tumhare paas ye hona chahiye:

* Design system (colors, fonts, spacing)
* Component library
* Homepage wireframe (premium)
* All page templates (UI level)
* Mobile-first layouts
* Visual hierarchy
* CTA strategy
* UI guidelines for developer

---

# 3.1 DESIGN PHILOSOPHY (VERY IMPORTANT)

Tumhari website ka tone kya hoga?

👉 **Premium + Trust + Discipline + Modern**

### Avoid:

* flashy colors
* clutter
* random animations
* cheap templates

### Target:

* cadet college style
* military discipline feel
* structured layout
* high trust

---

# 3.2 COLOR SYSTEM

Original site ke colors ko base lo, phir polish karo.

## Define:

* Primary color (school identity)
* Secondary color
* Accent color (buttons)
* Neutral colors (background, text)

## Example structure:

* Primary: Deep Blue / Green (authority)
* Accent: Gold / Orange (CTA)
* Background: White / light gray
* Text: dark gray / black

---

# 3.3 TYPOGRAPHY SYSTEM

Define:

* Heading font (strong, bold)
* Body font (readable)
* Line spacing
* Font sizes hierarchy

## Example:

* H1: bold, large (hero)
* H2: section title
* H3: subsection
* Body: clean readable

👉 Consistency bohat important hai

---

# 3.4 SPACING & GRID SYSTEM

Define:

* section spacing
* container width
* padding rules
* mobile margins

👉 Is se design clean aur premium lagta hai

---

# 3.5 COMPONENT SYSTEM (REUSABLE BLOCKS)

Tumhari site components se banegi.

## Must Components:

* Navbar
* Footer
* Hero section
* Section title block
* Cards (news, faculty, features)
* CTA blocks
* Image + text blocks
* Stats/counters
* Forms
* Buttons (primary/secondary)

👉 Har component reusable hona chahiye
